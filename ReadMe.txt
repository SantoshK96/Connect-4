Connect 4:
A simple implementation of the popular connect-four game using node & react.
author : Santosh Kumar Patra
Frontend using react.js
Backend using Node.js ,Express

Instruction To start the App:
	1.The following directory contains two separate folders.
	2.frontend logic can be found in connect4/ directory.
	3.To start the frontend app, run the below commands.
		cd ./connect4/
		npm start
	4.To start the Backend service, run the below commands.
		cd ./ gameService/
		npm start
